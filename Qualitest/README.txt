#Assignment - Qualitest

#Framework
Below code developed using java8, selenium, cucumber, Maven and jUnit.

## External dependencies

For this project to run, you would need to install below 3 dependencies on your machine:
- **[Java 8]** (as the core programming language)
- **[Maven -3.8.6]** (for dependency management)
- **[Google Chrome latest version]** (browser to run your tests)

## Getting Started
For easiest way to getting started, extract this project and open it from eclipse.
> Then Do a dry run on test in : /Qualitest/src/test/java/TestRunner/testRunner.java  -->RunsAs->jUnit
From cmd : cd projectpath
			mvn test  

##Reports:
Three types of Reports are generated
html:target/HtmlReports/report.html", "json:target/JsonReports/report.json","junit:target/JUnitReports/report.xml"
